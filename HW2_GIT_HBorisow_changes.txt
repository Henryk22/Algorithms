2.Как вернуть файл из индекса в рабочий каталог?

Чтобы вернуть файл из индекса Git в рабочий каталог (отменить его индексацию), 
можно использовать команду git reset с параметром --mixed. Например:

git reset --mixed <имя файла>
где <имя файла> - это имя файла, который нужно вернуть в рабочий каталог.

Эта команда отменит индексацию файла, но оставит изменения в файлах. 

Если нужно удалить изменения и вернуть файл в состояние последнего коммита, 
можно использовать параметр --hard вместо --mixed. Например:

git reset --hard <имя файла>

1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?


2. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

3. Как получить на локальный компьютер изменения с github.com

Какую информацию нам дают команда :
git status
git log
git remote -v 

===============
Решение:

1. Чтобы скопировать репозиторий на свой компьютер из GitHub, нужно сделать следующее:
Открыть страницу репозитория на GitHub
Нажать на кнопку "Code" (зеленая кнопка) в правом верхнем углу
Скопировать ссылку на репозиторий (лучше SSH)
Открыть терминал на своем компьютере
Перейти в директорию, где вы хотите сохранить репозиторий
Выполнить команду git clone <ссылка на репозиторий>

2. Чтобы создать репозиторий на GitHub из локальной папки на компьютере, нужно сделать следующее:
Создать локальный репозиторий с помощью команды git init в нужной директории
Добавить файлы в репозиторий с помощью команды git add . или git add <название файла>
Создать коммит с помощью команды git commit -m "Название коммита"
Создать репозиторий на GitHub и скопировать ссылку на него
Добавить ссылку на удаленный репозиторий с помощью команды git remote add origin <ссылка на репозиторий>

Отправить локальный репозиторий на GitHub с помощью команды git push -u origin master (для первой отправки) или git push origin master (для последующих отправок)

3. Чтобы получить изменения с GitHub на локальный компьютер, нужно сделать следующее:
Открыть терминал на своем компьютере
Перейти в директорию репозитория с помощью команды cd <название директории>
Выполнить команду git pull origin master для получения изменений с GitHub


Команда git status показывает текущее состояние репозитория (изменения, которые были внесены, но еще не зафиксированы, изменения, которые были зафиксированы, но еще не отправлены на GitHub и т.д.)

Команда git log показывает историю коммитов в репозитории.

Команда git remote -v показывает список удаленных репозиториев, связанных с локальным репозиторием.
